PromiseMix

Make Promises Talk!

A cleaner, more readable way
to chain your Promises.

From this...

                      Promise.resolve('Jade')
    .then(user => {
      return Promise.resolve(['Buck', 'Felix'])
        .then(pets => {user, pets});
  })...

...To this !

                      Promise.aggregate({
    user: Promise.resolve('Jade'),
    pets: Promise.resolve(['Buck', 'Felix'])
  })...
