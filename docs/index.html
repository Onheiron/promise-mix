Make Promises Talk!

A cleaner, more readable way
to chain your Promises.

From this...

                      Promise.resolve('Jade')
    .then(user => {
      return Promise.resolve(['Buck', 'Felix'])
        .then(pets => {user, pets});
  })...

...To this !

                      Promise.aggregate({
    user: Promise.resolve('Jade'),
    pets: Promise.resolve(['Buck', 'Felix'])
  })...

Compact

Zero dependencies, just a few files, no duplicate code.

Readable

PromiseMix makes it easier to fallow what's going on in a Promise chain.

Tested

Install

  $ > npm install promise-mix

Use

  require('promise-mix');
